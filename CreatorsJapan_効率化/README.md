# Creators Japan 業務効率化ポータル

Creators Japan向けの業務効率化ポータルシステム。
Cloudflare認証による管理者/クライアント専用ダッシュボードを提供。

## 技術スタック

- **フロントエンド**: React + Vite + Tailwind CSS
- **ホスティング**: Cloudflare Pages
- **認証**: Cloudflare Access
- **バックエンド**: Cloudflare Workers
- **データベース**: Cloudflare D1
- **ランニングコスト**: 0円/月

## MVP機能

### 1. ブログ記事報告
- 管理者がURLを入力 → OGP自動取得（タイトル、画像、公開日）
- クライアントがダッシュボードで一覧確認

### 2. GA4レポート自動化
- GA4 APIで2サイト分のデータを自動取得
- PV、ユーザー数、人気記事等を表示
- 月次でデータ蓄積

### 3. Search Consoleレポート
- 検索クエリ（どんなキーワードで検索されたか）
- 表示回数・クリック数・CTR・平均順位
- GA4と組み合わせたSEO分析

### 4. Discord自動投稿
- 毎月1日に所定チャンネルへレポート自動投稿
- サイト別に投稿先を分離
