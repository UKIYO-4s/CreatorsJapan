新規でGA4プロパティを作成いただく必要があります。
作成手順が必要でしたらお知らせください。

---

### 2. サービスアカウントへの権限付与

ポータルからGA4/Search Consoleのデータを自動取得するため、
以下のサービスアカウントに閲覧権限を付与してください。

**サービスアカウントメール:**
```
（GCP作成後に記載）
```

#### GA4での設定手順
1. GA4管理画面にログイン
2. 管理（左下の歯車）→ プロパティのアクセス管理
3. 右上の「+」→「ユーザーを追加」
4. 上記メールアドレスを入力
5. ロール:「閲覧者」を選択
6. 「追加」をクリック

※ 公式サイト・サロンサイト両方で設定をお願いします

#### Search Consoleでの設定手順
1. [Search Console](https://search.google.com/search-console) にログイン
2. 対象プロパティを選択
3. 設定 → ユーザーと権限
4. 「ユーザーを追加」
5. 上記メールアドレスを入力
6. 権限:「フル」を選択
7. 「追加」をクリック

※ 公式サイト・サロンサイト両方で設定をお願いします

| 用途 | チャンネル名 | Webhook URL |

#### Webhook作成手順（必要な場合）
1. Discordで対象チャンネルの設定を開く
2. 連携サービス → Webhook
3. 「新しいウェブフック」
4. 名前を設定（例: 月次レポートBot）
5. 「ウェブフックURLをコピー」
