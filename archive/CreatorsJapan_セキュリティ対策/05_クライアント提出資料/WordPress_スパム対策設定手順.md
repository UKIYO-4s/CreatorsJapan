# WordPress スパムコメント対策 設定手順書

## 現状の問題

2025年12月8日以降、カジノ・ギャンブル関連のスパムコメントが7件投稿されています。
今後の同様のスパムを自動ブロックするため、以下の設定をお願いいたします。

---

## 設定手順

### Step 1: WordPress管理画面にログイン

管理者アカウントでログインしてください。
（※この設定は管理者権限が必要です）

---

### Step 2: ディスカッション設定を開く

**方法A: メニューから**
1. 左側メニューの「設定」にマウスを乗せる
2. サブメニューから「ディスカッション」をクリック

**方法B: 直接アクセス**
```
https://salon.creators-jp.com/wp-admin/options-discussion.php
```

---

### Step 3: コメントブロックリストを見つける

ページを下にスクロールし、「コメントブロックリスト」という項目を探してください。

> この欄に入力した単語がコメント本文、投稿者名、URL、メールアドレス、
> IPアドレス、ブラウザーのユーザーエージェントに含まれる場合、
> そのコメントはゴミ箱に入れられます。

---

### Step 4: ブロックキーワードを追加

以下のキーワードをコピーして、テキストエリアに貼り付けてください：

```
casino
slot
gambling
betting
poker
roulette
jackpot
winwin
jl9casino
b52fun
a7xmexico
11xplayprologin
jljl77
ph799
.click
.info
```

**補足：**
- 1行に1キーワード
- 既存の内容がある場合は、その下に追加
- 大文字・小文字は区別されません

---

### Step 5: 設定を保存

ページ最下部の「変更を保存」ボタンをクリック

---

## 追加推奨設定

同じページ内で以下の設定も確認・変更をお勧めします：

### コメント承認設定
- [ ] 「コメントの投稿者の名前とメールアドレスの入力を必須にする」→ **有効**
- [ ] 「すでに承認されたコメントの投稿者のコメントを許可し、それ以外のコメントを承認待ちにする」→ **有効**

### リンク制限
- [ ] 「コメント内で許可するリンクの数」→ **1** または **2** に設定
  （スパムは大量のリンクを含むことが多いため）

---

## 確認方法

設定後、「スパム」タブを確認すると、ブロックされたコメントが自動的に振り分けられます：

```
https://salon.creators-jp.com/wp-admin/edit-comments.php?comment_status=spam
```
