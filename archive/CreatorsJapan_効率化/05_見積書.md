## 簡易版（MVP）詳細

### 価格: ¥30,000（税別）

**含まれる機能:**

| 機能 | 詳細 |
|------|------|
| Cloudflare認証 | 管理者/クライアント 2ロール |
| ポータルUI | レスポンシブ対応ダッシュボード |
| ブログ記事報告 | URL入力で自動情報取得・一覧表示 |
| クライアント閲覧 | 投稿された記事の確認 |
| **GA4自動レポート** | 月次でPV・ユーザー数等を自動取得・表示 |
| **Search Consoleレポート** | 検索クエリ・CTR・順位等を自動取得・表示 |
| **Discord自動投稿** | 毎月1日に所定チャンネルへ自動投稿 |

**技術仕様:**

| 項目 | 内容 |
|------|------|
| ホスティング | Cloudflare Pages |
| 認証 | Cloudflare Access |
| データベース | Cloudflare D1 |
| 外部連携 | GA4 Data API, Search Console API, Discord Webhook |
| ランニングコスト | 0円/月 |

**納品物:**
- ポータルサイト（デプロイ済み）
- 管理者アカウント設定
- クライアントアカウント設定
- GA4 API連携設定
- Discord Webhook設定
- 簡易マニュアル

**必要な情報（クライアントから）:**
- GA4 プロパティID（2サイト分）
- GA4 管理者権限 or サービスアカウント追加許可
- Search Console プロパティへのアクセス権限
- Discord Webhook URL（投稿先チャンネル）

**納期目安**: 契約後 2週間
